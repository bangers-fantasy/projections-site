2025-26 Fantasy Hockey Projection Report Card

How 11 public projection sets, Bangers Fantasy Hockey included, plus a Wisdom of the Crowd benchmark (the simple average of them all), did against the 2025-26 season. Every projector's raw category numbers become one fair overall value (a roto z-score, so no projector's own formula gets home-field advantage), then get scored on how well the resulting draft order matched reality. Everything is measured per full season, so injuries aren't punished (in head-to-head, a hurt player gets replaced). The headline compares the 9 entries that project all 7 value categories, on 270 shared players.

Position-adjusted: value over replacement (VORP)

Plain value is position-blind, and drafting is not. A scarce elite defenseman is worth more than his raw totals, because the drop-off to a replacement D is steep. This re-scores everyone as value over the last starter in a standard Yahoo 12-team league (6 F / 4 D per team, so replacement is the 72nd forward and 48th defenseman), which lifts scarce positions up the board. If the leaderboard survives this, the original result was measuring real value and not a scoring illusion. It survives.

The optimism check: who runs hot on their own board

Over-projection is the expensive error. You draft the bust immediately, in every draft. Under-projection only costs you a late gem in deep leagues. Does avoiding busts actually win? Testing the hypothesis

The pool-aid creators suspected the strongest lists won by drafting fewer over-projected players rather than by making more correct calls, but flagged it as unproven. We can test it directly. Across the 8 full-category projectors, does over-projection error predict draft utility more than total error does? (More negative means that kind of error hurts drafting more, and the hypothesis predicts over-projection should be the strongest negative.)

over-projection → draft utility
+0.40
under-projection → draft utility
-0.47
total error → draft utility
-0.51

In 2025-26 the hypothesis doesn't hold. Total accuracy is the best predictor (-0.51, so less error means better drafting), and under-projection hurts at least as much as over-projection. The sign on over-error is even slightly positive. Two honest reasons it differs from the 2023-24 finding: our per-82 framework strips out the injury-driven over-projection that made busts so costly there, and this season's elite tier was predictable (the stress test above), so over-reaching on consensus stars wasn't punished. With n=8 this is directional rather than definitive, but it's a real, testable answer where the source only had a hunch.

Method

  • Neutral yardstick. Draft utility is scored on roto z-score value, not any projector's own formula, so no source gets a home-field advantage. The Bangers league formula is run alongside only as a check.
  • Raw submissions. Each projector's published columns are used as given (no derived proxies), pro-rated to 82 games; sources missing a category are graded only where they supplied it.
  • Shared reference set per category. Every source is graded on the same players, so a deep-coverage source isn't punished for projecting extra fringe players.
  • Bias, skill and significance, not one blended number. Signed bias catches league-wide misses, skill compares against a naive model, and paired bootstraps flag which gaps are real versus noise.
  • All per-82 rate; injuries normalized out of both sides (games played deliberately not graded).
